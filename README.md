# PhotoShare

2인용 공유 사진첩. Apple 사진첩 스타일 그리드 + 전체화면 뷰, 업로드 시 가리기/블러/태그/캡션 일괄 적용, 태그 필터 및 정렬, EXIF 메타데이터 자동 파싱.

## 실행

```bash
cp .env.example .env
# .env 에서 사용자 이름/비밀번호, SESSION_SECRET 수정
docker compose up -d --build
```

브라우저에서 http://localhost:3018 접속.

## 갤럭시(안드로이드) 웹앱 설치

1. Chrome에서 PhotoShare를 연다.
2. 메뉴(⋮) → **홈 화면에 추가** 또는 **앱 설치**를 선택한다.
3. 설치 후 홈 화면 아이콘으로 실행하면 단독 앱(standalone)처럼 동작한다.

## 아이폰(iOS) 웹앱 설치

1. Safari에서 PhotoShare를 연다.
2. 하단 공유 버튼(□↑)을 누른다.
3. **홈 화면에 추가**를 선택한다.
4. 홈 화면 아이콘으로 실행하면 브라우저 UI 없이 웹앱 형태로 동작한다.

참고:
- 설치 프롬프트/앱 설치는 HTTPS 환경에서 가장 안정적으로 동작한다.
- `localhost`는 개발용 예외로 동작하지만, 휴대폰에서 PC IP로 접속할 때는 HTTPS 구성이 필요할 수 있다.

## 업로드 UX

- 파일을 드래그앤드롭 하거나, 파일 선택 대화상자로 다중 선택, 또는 클립보드에서 붙여넣기
- 업로드 전 썸네일 프리뷰 화면에서 항목별 또는 일괄로:
  - **가리기**: 올린 사람에게만 보임 (상대방 목록·원본·썸네일 모두 차단)
  - **바로보이지 않기 (블러)**: 모든 사용자에게 썸네일 블러 처리, 클릭하면 원본
  - **캡션**
  - **태그**
- 원본 크기 제한 없음 (스트리밍 업로드)

## 사진 보기

- Apple 사진첩 스타일 정사각 그리드
- 사진 클릭 시 전체화면 (←/→ 네비게이션, ESC로 닫기, `i`로 정보 패널 토글)
- 정보 패널: 캡션 편집, 파일 형식, 촬영/원본 생성/원본 수정 시각(가능한 경우), 카메라, 찍은 사람, GPS 위치 (OSM 링크), 해상도, 파일 크기, 태그
- 사진 상세 진입 시 이미지 메타데이터 자동 재파싱, 정보 패널의 **정보 재파싱** 버튼으로 수동 재파싱 가능
- **블러 토글은 누구나 가능** (요구사항). 가리기/캡션/태그/삭제는 올린 사람만.

## 정렬 & 필터

- 원본생성시각 / 최신순 / 오래된순 / 큰 크기순 / 작은 크기순
- 태그 다중 선택 → AND 필터
